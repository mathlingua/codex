Logic

Propositional (or zeroth-order), predicate (or first-order), and second-order logic form the logical basis for mathematics.

The logical constructs needed to describe mathematics are already provied by MathLingua. Commands are defined below that expose those underlying logical constructs.

First, a proposition is, informally, a statement, and is the fundamental object in propositional logic, and hence first and second order logic.

Unlike the normal usage of the word proposition in mathematics to mean a "little theorem" (that is true), a proposition is a statement that can be either true or false.
